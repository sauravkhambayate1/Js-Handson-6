4. Reverse a Number.

Write a program which takes a number N as input from the user and You need to reverse the number.

    var Reverse_Number = (N) => 
{
  var z=N
  var reverse = 0;
while(z > 0)
{
    var digit = z % 10;
    reverse = (reverse * 10) + digit;
    z = parseInt(z / 10);
}
   return reverse
};
